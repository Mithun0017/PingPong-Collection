# ✨ Features

- 🏓 Classic 2-Player Gameplay
- ⚡ Smooth HTML5 Canvas Rendering
- 🎯 Real-time Collision Detection
- 🚀 Increasing Ball Speed after each hit
- 📊 Live Scoreboard
- 🔄 Reset Game Button
- 🎮 Keyboard Controls
- 📱 Lightweight and Fast
- 💻 Runs directly in the Browser
- 🧩 Clean and Beginner-Friendly Source Code

# ⚙️ Game Mechanics

- Ball starts from the center
- Random initial movement direction
- Paddle collision changes ball direction
- Ball speed increases after every successful hit
- Player scores when opponent misses
- Scoreboard updates automatically
- Reset button restarts the game
