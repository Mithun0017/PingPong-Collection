## 📖 About

**PingPong Collection** is an open-source repository containing multiple implementations of the classic Pong game.

The goal is to showcase game development concepts using different programming languages, libraries, frameworks, and engines while gradually increasing complexity from beginner-friendly projects to advanced AI-powered multiplayer games.

Whether you're learning game development or exploring different technologies, this repository has something for you.

---

# 🎯 Objectives

- Learn game development fundamentals
- Compare implementations across languages
- Practice object-oriented programming
- Explore game physics
- Build AI opponents
- Create multiplayer experiences
- Experiment with graphics and animations
